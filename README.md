# TBI Game

## Group Number
Group 10B

## Description
A p5.js game designed to foster understanding of Traumatic Brain Injury (TBI) through interactive gameplay mechanics. The game simulates common TBI-related challenges such as memory loss and sensory overload through mechanics like Memory Fade and Sensory Overload. Players must manage limited information, recall objectives, and regulate overload levels while completing tasks. 

The game emphasizes empathy and awareness by translating cognitive and sensory challenges into playable systems, rather than relying on stereotypes or narrative explanation alone. Recent updates include an active Calm Ability system, where players must strategically manage recovery instead of relying on passive safe zones, and environment-based visuals to better connect gameplay with real-world contexts.

---

## Setup and Interaction Instructions
1. Open the game via the GitHub Pages link, or open `index.html` in Google Chrome.
2. Press **ENTER** to start the game.
3. Use **WASD or arrow keys** to move the player.
4. Press **M** to briefly recall the objective when it fades.
5. Press **J** to activate the Calm Ability (if charges are available).
6. Press **L** to toggle Low Sensory Mode.
7. Press **R** to return to the title screen.
8. Press **N** to toggle map.

## Iteration Notes

### Post-Playtest
- Increased gameplay intensity by introducing more challenge and decision-making instead of purely passive mechanics.
- Redesigned Calm Zones into an active ability system (Calm Ability) to give players control over recovery.
- Adjusted visual clarity and UI to improve readability and reduce confusion during gameplay.

### Post-Showcase
- Plan to add moving obstacles or enemies to better represent unpredictability and pressure.
- Improve color contrast and visual hierarchy to avoid elements blending together.
- Expand environmental storytelling using more real-world inspired assets and contextual elements.
